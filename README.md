# Retail Sales Analytics Dashboard

## Project Overview
This project focuses on analyzing retail sales data using multiple data analytics and visualization tools including Excel, Power BI, Tableau, Python, and SQL. The goal of this project is to identify sales trends, regional performance, category contribution, and profit insights through interactive dashboards and business analysis.

---

# Tools & Technologies Used
- Excel
- Power BI
- Tableau
- Python
- SQL

---

# Project Objectives
- Analyze retail sales performance
- Identify top-performing regions and categories
- Track monthly sales trends
- Analyze profit contribution
- Build interactive dashboards
- Generate business insights for decision-making

---

---

# Key Business Insights
- Technology category generated the highest sales and profit.
- West region achieved the best sales performance.
- Sales peaked during November and December.
- Furniture category showed lower profitability compared to sales.
- Regional performance varied significantly across the dataset.
